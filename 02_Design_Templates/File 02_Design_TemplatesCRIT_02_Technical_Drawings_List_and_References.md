# CRIT Module 02: Technical Drawings List and Reference Templates

## Purpose
This document provides a standard list of technical drawings required for climate-resilient school construction and retrofitting. It supports engineers and works departments in preparing compliant drawings.

---

## Mandatory Drawings (Minimum Package)
Each school design must include:

### 1. Architectural Drawings
- Site plan with boundaries and access routes
- Floor plan(s) for classrooms and WASH blocks
- Elevation views (front, rear, side)
- Section view (roof + foundation)

### 2. Structural Drawings
- Foundation plan (plinth height specification)
- Column-beam reinforcement details
- Roof truss details and anchoring system
- Wall reinforcement details (seismic bands where applicable)

### 3. Drainage and Flood Protection Drawings
- Drainage layout map (internal and external)
- Water flow direction plan
- Soak pit / rainwater harvesting design (if applicable)
- Boundary protection/embankment design (where needed)

### 4. Heat Mitigation and Ventilation Drawings
- Roof insulation specification sheet
- Shading structures / veranda design
- Window placement for cross-ventilation

### 5. WASH Drawings
- Toilet layout (gender segregated)
- Handwashing station design
- Water tank / storage layout
- Septic tank / sewer connection plan

### 6. Accessibility Drawings
- Ramp slope and access pathway details
- Accessible toilet design
- Handrail and signage placement

---

## Recommended Reference Standards
Design and drawings should align with:
- UNICEF Safe Schools principles
- National/provincial building codes
- NDMA/PDMA disaster resilience guidelines (where available)
- Minimum WASH standards for schools

---

## Drawing Submission Checklist (for Approval)
- [ ] Site plan includes hazard mapping notes
- [ ] Plinth level defined and justified against flood history
- [ ] Drainage plan includes discharge points
- [ ] Roof anchoring system included
- [ ] Inclusive access features included
- [ ] WASH design includes separate girls’ toilets
- [ ] Cost estimate/BOQ aligned with drawings
