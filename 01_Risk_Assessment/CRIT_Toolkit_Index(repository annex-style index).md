# CRIT Toolkit Index (Repository Annex Guide)

## Repository Title
**Climate-Resilient School Infrastructure Toolkit (CRIT)**  
Developed by: **WUM Advisory Services**  
Focus: Climate adaptation and resilient school infrastructure planning, design, retrofitting, and implementation.

---

## Purpose of this Index
This index provides a structured overview of the CRIT repository modules and files. It is designed to support reviewers, partners, and implementing agencies in quickly navigating CRIT resources.

---

# MODULE DIRECTORY

## 📁 Module 01: Risk Assessment & Prioritization (`/01_Risk_Assessment`)
**Purpose:** Identify hazard exposure, assess school vulnerability, and prioritize schools for intervention.

### Files:
- **CRIT_01_Climate_Risk_Screening_Tool.md**  
  Climate hazard screening tool and composite risk scoring framework.

- **CRIT_01_School_Vulnerability_Assessment_Checklist.doc**  
  School infrastructure assessment checklist (site, structural, WASH, electrical, inclusion).

- **CRIT_01_Rapid_Assessment_Form_Field_Use.doc**  
  Rapid one-page field assessment tool for district teams.

- **CRIT_01_School_Risk_Prioritization_Scoring_Template.xlsx**  
  Excel-based vulnerability scoring and prioritization template.

---

## 📁 Module 02: Climate-Resilient Design Templates (`/02_Design_Templates`)
**Purpose:** Provide minimum climate-resilient design standards for new school construction and rehabilitation.

### Files:
- **CRIT_02_Modular_Climate_Resilient_School_Design_Guidelines.doc**  
  Design guidelines for flood, heat, storm, WASH, inclusion, and safety.

- **CRIT_02_Technical_Drawings_List_and_References.md**  
  Drawing requirements and reference list for engineers and works departments.

- **CRIT_02_Minimum_Resilience_Design_Standards_Table.xlsx** *(optional file recommended)*  
  Compliance matrix for design standards (to be added).

---

## 📁 Module 03: Retrofitting Guidelines (`/03_Retrofitting_Guidelines`)
**Purpose:** Provide practical retrofitting packages to upgrade existing schools for climate resilience.

### Files:
- **CRIT_03_Retrofitting_Manual_Flood_Heat_Storm.doc**  
  Retrofitting manual covering flood, heat, storm, WASH, electrical safety, and accessibility.

- **CRIT_03_Retrofitting_Bill_of_Quantities_Template.xlsx**  
  BOQ template for costing retrofit interventions.

- **CRIT_03_Site_Safety_and_Construction_Quality_Checklist.doc** *(optional file recommended)*  
  Construction safety and quality assurance checklist (to be added).
