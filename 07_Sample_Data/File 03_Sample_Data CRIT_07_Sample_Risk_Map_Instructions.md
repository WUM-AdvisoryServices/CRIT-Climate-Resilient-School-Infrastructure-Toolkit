# CRIT Sample Risk Mapping Instructions

## Purpose
This document provides guidance on generating simple school risk maps for prioritization and planning.

---

## Tools Recommended
- Google MyMaps (easy, free)
- QGIS (advanced GIS tool)
- Excel + Google Maps pinning

---

## Minimum Data Required
For each school:
- School name and EMIS code
- District and UC
- GPS coordinates
- Hazard exposure score (Module 01)
- Vulnerability score
- Composite risk score

---

## Step-by-Step (Google MyMaps)
1. Create a Google Sheet with columns:
   - School Name
   - Latitude
   - Longitude
   - Risk Level (Low/Medium/High/Critical)

2. Open Google MyMaps and create a new map.

3. Import the sheet and select Latitude/Longitude columns.

4. Use “Style by column” and choose Risk Level.

5. Export map link for sharing with education authorities.

---

## Suggested Data Sources (Pakistan Context)
- NDMA/PDMA hazard and flood maps
- Pakistan Meteorological Department (PMD) heatwave reports
- Satellite rainfall datasets (where available)
- District disaster management reports
- UNICEF situation reports (where accessible)
